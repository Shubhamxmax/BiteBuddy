# 🍔 Food Delivery App – Project README

## 📌 Overview

This is a **React-based Food Delivery App** where users can browse items, use a live search bar, and manage their cart. The focus is on smooth UI, state management, and responsiveness.

---

## 🚧 Challenges & Solutions

### 🔍 Active Search Bar

**Problem:** Real-time search caused unnecessary re-renders.
**Solution:** Optimized filtering and controlled input handling.

### 🔄 State Management (Redux)

**Problem:** Managing cart state across components was complex.
**Solution:** Used **Redux** for centralized and consistent state management.

### 🛒 Cart Functionality

**Problem:** Handling add/remove/update without bugs.
**Solution:** Created clear Redux actions and ensured instant UI updates.

### 🪟 Popup UI

**Problem:** Managing popup visibility smoothly.
**Solution:** Built a simple reusable popup component with clean state handling.

### 📱 Responsive Design

**Problem:** UI breaking on different screen sizes.
**Solution:** Used Flexbox, Grid, and media queries for responsiveness.

---

## 🛠️ Tech Stack

* React.js
* Redux
* CSS (Flexbox & Grid)
